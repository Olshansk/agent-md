# Proposed Changes Format

Use this exact structure when proposing updates to `AGENTS.md`.

## Proposed Changes to `AGENTS.md`

> **Summary:** X additions, Y modifications, Z removals

---

### Additions (X)

> **Section: [Section Name]**
>
> ```diff
> + New content being added
> ```

---

### Modifications (Y)

> **Section: [Section Name]**
>
> **Before:**
>
> ```diff
> - Old content
> ```
>
> **After:**
>
> ```diff
> + New content
> ```

---

### Removals (Z)

> **Section: [Section Name]**
>
> ```diff
> - Content being removed
> ```
>
> **Reason:** Why this is being removed

Then ask: **"Do you want me to apply these changes to `AGENTS.md`?"**
