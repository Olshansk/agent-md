# Skills Catalog

Available skills in this repository:

- `session-commit` — Captures session learnings and updates `AGENTS.md` with explicit approval
- `skills-dashboard` — Scrapes skills.sh and generates an interactive HTML dashboard of the ecosystem

Each skill directory follows the Agent Skills specification:

```text
<skill-name>/
  SKILL.md
  scripts/      # optional
  references/   # optional
  assets/       # optional
  commands/     # optional
```
